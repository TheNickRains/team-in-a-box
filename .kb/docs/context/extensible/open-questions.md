# Open Questions

Cross-domain questions that don't belong to a single agent. Agents surface questions here when work touches another persona's domain and a decision needs to be made before proceeding.

---

## Entry format

```
### YYYY-MM-DD — <question title>
**Raised by:** <agent name>
**Blocks:** <task id or description of what's blocked>
**Question:** <the specific question that needs an answer>
**Options considered:** <brief, optional>
**Resolved:** <date + decision — fill in when closed>
```
